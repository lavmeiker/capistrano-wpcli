# Capistrano::Wpcli

Simple Capistrano wrapper using WP-CLI

## Installation

Add this line to your application's Gemfile:

    gem 'capistrano-wpcli'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install capistrano-wpcli
